# 本次修改说明
1. translate.google.cn又用得了了，所以改成失败时才选用代理连接.com方式
2. 使用translate.google.com时需要配置好计算机代理功能
3. 如果计算机有socks代理等，请转为http/https代理,并监听到8123端口
4. 默认配置使用polipo，将socks转https，其默认端口号是8123
5. 可以使用其他，需自行配置
