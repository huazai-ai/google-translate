# 本次修改说明
1. translate.google.cn突然用不了了，所以改为.com
2. 因为用的.com的原因，需要使用代理功能才能正常使用
3. 如果计算机有socks代理等，请转为http/https代理 
4. 默认配置使用polipo，将socks转https，端口号是8123
5. 如果使用其他，请自行配置
6. 本次更新还修复了部分逻辑错误

# 功能
* **中英单词互译**
* **支持代理功能**
* **句子翻译功能**
* **提供URL可查看翻译详情**
